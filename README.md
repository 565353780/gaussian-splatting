# Gaussian Splatting

## Requirements

```bash
gcc-12 & g++-12
cuda-11.8
```

## Source

```bash
https://github.com/graphdeco-inria/gaussian-splatting
```

## Install

```bash
conda create -n gs python=3.11
conda activate gs
./setup.sh
```

## Run

```bash
python convert.py
python demo.py
python render.py
```

## Enjoy it~
